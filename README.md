SpringMVC Rest- Best Practices
=============================
*** Work in Progress ***

A reference app to implement REST Level 3 (HAETEOAS), concepts like Futures, asynchronous processing, 
Actors (Akka), etc using spring mvc framework 3.1.x.

Libraries used
- Spring 3.1.3
- JPA 2
- Hibernate 3.6
- H2
- Twitter Bootstrap
- JQuery
- Akka
